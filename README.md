## Bus Management Prototype

Prototype to solve girls transport problems, book ride and travel safely.

### Tech

- React framework (react-router)
- Tailwind css
- Motion
- R3F

### Tools

- Google Earth Studio for Hero video
- Blender for [3D model][1] Optimization
